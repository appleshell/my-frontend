---
title: 链表
---

多个元素组成的列表，但是存储的位置不连续，使用 next 指针连在一起。在 javascript 中，可以使用 Object 来实现链表

遍历链表, 链表的遍历需要借助指针，在代码中，通常会定一个变量

```js
const a = { val: "a" };
const b = { val: "b" };
const c = { val: "c" };
const d = { val: "d" };

a.next = b
b.next = c
c.next = d

let p = a
while(p) {
  console.log(p.val)
  p = p.next
}
```
